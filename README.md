# MovieDatabase-Sql-Project

This movie-related database is tailored to cater to the curiosity of movie enthusiasts. By storing and organizing vast amounts of data about movies, actors, directors, genres, reviews, and ratings, it serves as a comprehensive resource for anyone passionate about films.
Imagine you're a movie lover eager to explore the cinematic world. 

List of tables in the movie database:
actor,
genres,
director,
movie,
movie_genres,
movie_direction,
reviewer,
rating,
movie_cast,
